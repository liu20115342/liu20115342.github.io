### 环境

> node 环境下使用 yarn 或 npm

### 新建一个博客

```bash
yarn new "My New Post"
# 或
npm run new "My New Post"
```

### 草稿页合并到发布页

```bash
yarn push draft "page name"
# 或
npm run push draft "page name"
```

### 启动 server 本地服务默认开启 debug draft

```bash
yarn s
# 或
npm run s
```

### 构建静态页

```bash
yarn b
# 或
npm run b
```

### 发布

```bash
yarn d
# 或
npm run d
```
